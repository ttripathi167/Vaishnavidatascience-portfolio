Projetos de Ciência de Dados

Programa de fidelidade de clientes - Cluster

Os programas de fidelidade dos clientes ganharam uma importância significativa no cenário empresarial atual, impulsionando o crescimento sustentável e a lucratividade. Ao promover conexões mais fortes com os clientes, esses programas levam a um aumento nas compras repetidas e níveis mais altos de engajamento.

Neste projeto, desenvolvi um modelo de segmentação de clientes usando técnicas de clusterização. Os resultados mostraram que um cluster específico, composto por apenas 11% dos clientes totais, gera impressionantes 50% da receita da empresa. Essas informações possibilitam a implementação de estratégias de marketing direcionadas, adaptadas a cada cluster de clientes, maximizando a eficácia do programa de fidelidade.

Foi criado um dashboard no Looker para implantar este projeto. Ele fornece informações sobre a empresa e os grupos de clientes gerados pelo modelo de clusterização. O painel funciona como uma plataforma centralizada para acessar insights e métricas relacionadas aos negócios e à segmentação de clientes.

Classificação - Learning to Rank

Neste projeto, eu desenvolvi um modelo de aprendizado de máquina para prever a probabilidade de clientes comprarem seguro de carro. O modelo foi treinado com dados coletados de clientes existentes que possuem seguro de saúde. Ao focar apenas nos clientes mais propensos a comprar seguro, a empresa pode reduzir o custo de fazer chamadas de vendas desnecessárias e melhorar sua lucratividade geral.

O produto final deste projeto é um documento do Google Sheets que está vinculado ao modelo de aprendizado de máquina atualmente em produção. Este documento fornece uma lista de clientes que provavelmente comprarão o seguro de carro, com base nas previsões feitas pelo modelo.

Como resultado deste projeto, a empresa obteve um aumento de 106,91% na lucratividade devido a economias de custos pela redução do número de chamadas de vendas desnecessárias.

Clientes em Churn e baixo ticket médio

Este projeto tem como foco abordar alguns dos desafios enfrentados pela empresa fictícia North Winds, incluindo questões como dados desorganizados, perda de clientes e baixo ticket médio.

Para ajudar a resolver esses problemas, foram utilizadas várias técnicas de Ciência de Dados, como organização do banco de dados, análise de perda de clientes, análises de cohort e RFM, segmentação de clientes com base em seu comportamento de compra, implementação de um sistema de recomendação e introdução de uma estratégia de Preço Elástico.

Através do uso dessas técnicas, foi possível fornecer informações valiosas e soluções para a North Winds, permitindo que eles tomem decisões orientadas por dados e melhorem sua lucratividade geral.

Regressão em Séries Temporais - Previsão de vendas

Este projeto tem como foco a previsão de vendas para as próximas 6 semanas da rede de farmácias Rossmann, utilizando modelos de Machine Learning. Os modelos foram treinados com base em dados históricos de vendas, e os resultados são entregues por meio de um chatbot no Telegram.

Cinco modelos diferentes foram implementados e o modelo XGBoost foi selecionado como o mais preciso. Para implantar o modelo em produção, foram utilizados o Flask e a nuvem do Heroku, e um chatbot no Telegram foi criado para que os usuários possam acessar as previsões de qualquer lugar.

Em geral, este projeto mostra o potencial do Machine Learning na previsão de vendas e destaca os benefícios de usar um chatbot para entregar os resultados de forma acessível.

Para obter insights valiosos sobre o mercado de moda, neste projeto realizo Web Scraping para coletar dados sobre produtos e preços de uma loja virtual de roupas. Projeto de Analise de Dados no Mercado Imobiliário

O projeto House Rocket tem como foco maximizar os lucros por meio de oportunidades de investimento em imóveis.
